# 2026년 CSS 최신 기법들

CSS는 매년 강력한 새 기능들이 추가되고 있습니다. 이제는 JavaScript 없이도 많은 것들을 구현할 수 있게 되었습니다.

## Container Queries

미디어 쿼리는 뷰포트 기준이지만, 컨테이너 쿼리는 부모 요소 크기를 기준으로 스타일을 적용합니다.

```css
.card-wrapper {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 1fr 2fr;
  }
}
```

## :has() 셀렉터

부모 선택자라고도 불리는 `:has()`는 CSS의 게임 체인저입니다.

```css
/* 이미지를 포함한 카드에만 스타일 적용 */
.card:has(img) {
  padding: 0;
}

/* 체크박스가 체크된 경우 레이블 스타일 변경 */
label:has(input:checked) {
  font-weight: bold;
  color: blue;
}
```

## @layer로 CSS 캐스케이드 관리

레이어를 사용하면 스타일 우선순위를 명확하게 관리할 수 있습니다.

```css
@layer base, components, utilities;

@layer base {
  h1 {
    font-size: 2rem;
  }
}

@layer components {
  .hero h1 {
    font-size: 3rem;
  }
}

@layer utilities {
  .text-xl {
    font-size: 1.25rem;
  }
}
```

## CSS Nesting

이제 CSS 네이티브에서도 네스팅이 가능합니다.

```css
.nav {
  display: flex;
  gap: 1rem;

  & a {
    color: inherit;
    text-decoration: none;

    &:hover {
      text-decoration: underline;
    }
  }
}
```
